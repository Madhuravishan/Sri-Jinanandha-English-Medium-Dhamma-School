A modern, single-page application (SPA) built for the ** Sri jinanandha English Medium Daham School**. This website serves as the digital front door for the school, providing parents and students with easy access to announcements, syllabus details, and contact information.

## 🌟 Features

* **Modern UI/UX:** Clean, respectful, and peaceful design palette tailored for a Buddhist educational institution.
* **Fully Responsive:** Mobile-first design that scales perfectly across phones, tablets, and desktop displays.
* **Smooth Animations:** Integrated with AOS (Animate On Scroll) for beautiful, fluid reveal animations as the user navigates the page.
* **Dynamic Sections:** Includes a Notice Board, Class Information, Image Gallery, and a Contact form UI.
* **Lightweight:** Built without heavy frontend frameworks for maximum speed and immediate loading.

## 🛠️ Technology Stack (Current)

This project is currently in **Phase 1** (Frontend Prototype). It was built to be lightweight and easily hosted via GitHub Pages.

* **Markup:** Semantic HTML5
* **Styling:** Tailwind CSS (via CDN)
* **Interactivity:** Vanilla JavaScript (ES6)
* **Animations:** AOS (Animate On Scroll) Library
